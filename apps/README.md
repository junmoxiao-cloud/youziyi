# YouZiYi (游子衣) - Apps Monorepo

这是《游子衣》项目的核心业务代码存放区。本项目采用多端协同的架构，以满足不同场景和用户的需求。

## 目录结构说明

- `harmony/`: 鸿蒙原生应用端（基于 ArkTS），提供原生的流畅体验与深度的系统整合。
- `miniapp/`: 微信小程序端，主打 MVP 快速落地与适老化交互，方便老年人通过极低门槛使用（大字号、高对比度）。
- `server/`: 后端服务层，处理数据存储、业务逻辑、状态同步以及超时无互动预警机制。
- `web/`: 网页端，主要用于答辩大屏展示、复杂的交互动效（如 3D 城市天气瓶渲染）。

## 开发指南

在开始各端开发前，请进入相应的目录，并参考其内部的配置文件与文档。
对于跨端的公共依赖或接口定义，后续可能会提取至顶层进行复用。
