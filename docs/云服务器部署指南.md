## 部署方式

### 方式一：GitHub 拉取（推荐标准流程）

适用于代码已推送到 GitHub 的情况。

```bash
ssh root@115.29.231.222
cd /var/www/youziyi/app
git pull origin main

# 安装依赖
cd packages/types && npm install && npm run build
cd ../../apps/server && npm install
cd ../web && npm install

# 构建后端
cd ../server && npm run build

# 构建前端
cd ../web && npm run build

# 数据库迁移（如有更新）
cd ../server && npx prisma migrate deploy

# 重启后端服务
cd ../server && pm2 restart all
```

### 方式二：本地上传（快速热修复）

适用于本地修改后需要快速验证、或 GitHub 网络不可用的情况。

**1. 本地构建前端**

```bash
cd apps/web
npx vite build --emptyOutDir
```

**2. SCP 上传到服务器**

```bash
scp -r "apps/web/dist/*" root@115.29.231.222:/var/www/youziyi/app/apps/web/dist/
```

**3. 修复文件权限（关键！）**

SCP 上传的文件默认归属 `root`，Nginx 以 `nginx` 用户运行，**必须修复权限否则页面会空白**。

```bash
ssh root@115.29.231.222 "chown -R nginx:nginx /var/www/youziyi/app/apps/web/dist/ && chmod -R 755 /var/www/youziyi/app/apps/web/dist/"
```

## Nginx 配置

### 前端站点（www.youziyi.com）

文件：`/etc/nginx/conf.d/www.youziyi.com.conf`

```nginx
server {
    listen 80;
    server_name www.youziyi.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl;
    server_name www.youziyi.com;

    ssl_certificate /var/www/youziyi/ssl/www.youziyi.com.crt;
    ssl_certificate_key /var/www/youziyi/ssl/www.youziyi.com.key;

    root /var/www/youziyi/app/apps/web/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

### API 反代（api.youziyi.com）

文件：`/etc/nginx/conf.d/api.youziyi.com.conf`

```nginx
server {
    listen 80;
    server_name api.youziyi.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl;
    server_name api.youziyi.com;

    ssl_certificate /var/www/youziyi/ssl/api.youziyi.com.crt;
    ssl_certificate_key /var/www/youziyi/ssl/api.youziyi.com.key;

    location / {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### 常用命令

```bash
# 测试配置
nginx -t

# 重载配置
systemctl reload nginx

# 查看状态
systemctl status nginx
```

## 后端服务（PM2）

```bash
# 启动/重启
pm2 start /var/www/youziyi/app/apps/server/dist/index.js --name youziyi-api
pm2 restart youziyi-api

# 查看状态
pm2 status
pm2 logs youziyi-api

# 开机自启
pm2 startup systemd
pm2 save
```

## 注意事项

1. **SCP 上传后务必修复权限**，否则 Nginx 无法读取 JS/CSS 导致页面空白
2. **本地 `vite preview` 不会启用代理**，测试 API 需使用 `vite dev` 或访问服务器
3. **数据库迁移**请在服务器上执行，不要直接操作生产数据库文件
4. **建议定期备份** `/var/www/youziyi/data/prod.db`
