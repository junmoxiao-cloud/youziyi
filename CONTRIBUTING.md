# 贡献指南

欢迎参与《游子衣》项目的开源协作。

## 开发与提交

- 请遵循项目内的工程约定与接口规范（见 `.trae/rules/`）。
- 提交信息建议使用 Conventional Commits（例如 `feat(web): add weather bottle`、`fix(server): handle null userId`）。
- 提交前请确保：
  - 通过本地 lint / typecheck（如工程已配置）
  - 关键功能有可复现的说明或测试

## 安全与隐私

- 严禁提交任何敏感信息，包括但不限于：`.env*`、密钥/证书、个人隐私数据、真实服务器参数、身份证件/照片等。
- 如需配置，请使用示例文件并在本机创建私有配置。

## 提交内容边界

本仓库以“可公开源码”为边界，不包含：

- 真实生产环境配置与凭据
- 服务器/域名/备案等敏感材料
- 个人隐私文件

如你不确定某文件是否可公开，请在提交前先移除并发起讨论。
